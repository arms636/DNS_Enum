# DNS_Enumeration
A simple shell script to collect DNS informations of a domain.

# Current Feature
- Can collect records of :

 1.IPv4
 
 2.IPv6
 
 3.CNAME
 
 4.MX
 
 5.TXT
 
 ## Usage
 ```
 $ git clone https://github.com/arms636/DNS_Enum.git
 $ cd DNS_Enum
 $ chmod 755 dns_recon.sh
 $ ./dns_recon.sh
